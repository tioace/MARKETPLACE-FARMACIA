 # MARKETPLACE-FARMACIA

Este repositório contém o front-end do projeto ClickFarma — um marketplace para farmácias, profissionais, consumidores e entregadores. Como requisito parcial para a aprovação no II semestre letivo da disciplina **Desenvolvimento Front-End**, no curso de graduação de **Ciência da Computação da Universidade Católica de Brasília**.

## Estrutura principal do projeto

Principais pastas e arquivos (resumo):

- `index.html` — página principal
- `assets/` — estilos, imagens e recursos
  - `css/` — estilos globais e específicos por página
  - `img/` — imagens e logo (`ClickFarmaLogo.jpg`)
- `paginas/` — páginas organizadas por categoria (ex.: `entregador/`, `consumidor/`, `cad-log/`, `dashboard/`)

## Como executar o projeto localmente

1) Usar o Live Server do VS Code:
	- Instale a extensão "Live Server" no VS Code.
	- Abra a pasta do projeto no VS Code e clique em "Go Live" no canto inferior direito.
	- O site será servido em `http://127.0.0.1:5500` (ou porta configurada).
